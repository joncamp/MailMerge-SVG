MailMerge‑SVG

Mail-merge a CSV onto an SVG template — tile it into a laser-ready grid, or make one page per row (certificates, badges). All in your browser; nothing is uploaded.

Load a template and a CSV (or click Load example) to get started.
Tip: for grid mode the template needs a tile group labelled inkscape:label="Nametag" (or Tile/Cell) with a cut outline labelled Nametag Border. For individual mode any whole-page SVG works. Either way, put {{TOKEN}} placeholders in the artwork and CSV column names are matched to them automatically. Host this file free on GitHub Pages, a gist (via htmlpreview), or anywhere static.
