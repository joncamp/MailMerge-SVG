Tip: for grid mode the template needs a tile group labelled inkscape:label="Nametag" (or Tile/Cell) with a cut outline labelled Nametag Border. For individual mode any whole-page SVG works. Either way, put {{TOKEN}} placeholders in the artwork and CSV column names are matched to them automatically. For certificates, add a hidden Inkscape layer labelled Backdrop (e.g. an image of the pre-printed stock) to preview alignment — it's shown only on screen, never in downloads.
